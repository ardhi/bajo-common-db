Common database for [Bajo](https://github.com/ardhi/bajo)

## Installation

Goto your ```<bajo-base-dir>``` and type:

```bash
$ npm install bajo-common-db
```

Now open your ```<bajo-data-dir>/config/.plugins``` and put ```bajo-common-db``` in it
